# 🛠️ Obsidian Vault - Personal Automation Utilities

[![Obsidian](https://img.shields.io/badge/Powered%20by-Obsidian-purple?logo=obsidian&logoColor=purple)](https://obsidian.md/)

[![Maintenance](https://img.shields.io/badge/Maintained%20by-Individual-blue)]()

这个仓库主要用于备份和管理我在 Obsidian 知识库中构建的一系列**通用配置、数据字典以及第三方插件（Templater, QuickAdd, Dataview）调用的高效工具脚本**。

本仓库旨在通过高度解耦、全配置驱动的代码片段，消除 Obsidian 日常使用中的机械化重复劳动，打造流程自动化的极客效率库。

## 📂 自动化工具模块划分

随着库功能的完善，脚本与配置文件按不同的功能域（Scope）进行前缀命名空间划分。目前只有 `para`

## ⚙️ 核心技术栈与调用规范

本仓库内所有脚本都遵循高复用、低耦合的设计原则，完美适配以下 Obsidian 明星插件：

* **Templater (User Script Functions)**
* **QuickAdd (Macro Execution)**
* **Dataview / DataviewJS**

## 🚀 部署与同步指引
查看对应模块下的 `readme.md` 文件

### 1. 脚本及配置文件放置

* **数据字典**：建议统一放置在 模板文件夹 下的 `data` 文件夹下
* **JS 脚本**：建议统一放置在 模板文件夹 下的 `scripts` 文件夹下
